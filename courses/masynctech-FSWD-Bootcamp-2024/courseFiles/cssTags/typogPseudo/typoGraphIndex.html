Typographical Pseudo Class Selector (::)

Return to CSS Tags Index

  • Used to style specific text parts
  • Target first letter, first line, etc.
  • Enhances typography and layout in CSS
    Examples:
  • ::first-letter
    • Used for chapter starts to get the readers attention
    • Targets the first letter of a block-level element
    • Style for "Drop Caps" or special effects
  • ::first-line
    • Used for headlines, quotes, and distinctive first line styles
    • Targets the first line of a block-level element
    • Style the first line, regardless of line breaks
    Best Practices
  • These are "Block" level elements
  • They can adjust font-size, line-height, and aligned styles with content
  • They maintain readability and accessability
    Limitations
  • May not work on all browsers
  • inheritance: ::first-letter & ::first-line inherit selectively, be precise
