CSS Descendent Combinator Selector (space)

Return to CSS Tags Index

Select a tag based on it's relationship to the root tag

eg. [ root tag ]
               [ nested tag]

would be referenced in the stylesheet with root[space]selected tag: eg. [ root(space)nestedTag {} ]

The nested tag can be a tag, class, or id.
