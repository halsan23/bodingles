CSS General Sibling Combinator Selector (~)

Return to CSS Tags Index

Targets all elements at the same level, with the same parent, which allows
you to target ALL matching siblings under the same parent tag.

Use this tag when you want to target all elements, under a specific "parent" tag,
but on the same level as the siblings.
