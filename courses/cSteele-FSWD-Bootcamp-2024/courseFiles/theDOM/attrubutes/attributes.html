Element Attributes

Most elements on a webpage will have some attributes associated with them. An [a] tag for example will have a src attribute. Other elements may have attributes such as id, class, title, or style. With JavaScript you can target these attributes.

We know we can use getElement() and querySelector() to return an element, or querySelectorAll() to return multiple elements. For example, if we call document.querySelector('p'); on this webpage, it would return:

  • < p>Colt Steele< /p>
because querySelector() will return the 1st instance of the element ([p] in this example) and "Colt Steele" is the 1st [p] on this page.

However, if we call querySelectorAll('p'); on this page, it would return:

  • NodeList(7) [p, p, p, p, p, p, p]
because there are seven [p] tags on this page. We can also expand the node list by clicking the down arrow and it list's all of the [p] tags. We can then further expand each individual [p] tag to see it's own properties and attributes.

We can also call querySelectorAll('p')[5]; to list the the 4th [p] tag (because it starts counting at 0) where you can then expand to see it's individual properties and attributes.
