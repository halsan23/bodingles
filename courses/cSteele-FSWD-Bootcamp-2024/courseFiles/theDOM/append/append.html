Adding HTML Items

Adding new HTML elements is pretty straight forward, but there are multiple steps.

We want to add a NEW H6 element to this webpage, but we want to add it to the end of THIS paragraph, which is contained within an [article] element. So we need to select the 1st [article] element and assign it to a variable:

  • const paragh = document.querySelector('article');
    • selects the 1st article element on the page and assigns it to the variable paragh
Now we can create the NEW h6 element:
  1. create a new element and assign it to a variable
    • const newEle = document.createElement('h6');
      • creates a new empty H6 element.
  2. add info to the new element, such as text, src, etc.
    • newEle.innerText = 'I am the New H6 Element';
      • the new empty H6 element now has some text.
  3. append the new element to the end of the specified document location, in this case, to the end of the 1st article element of the page.
    • paragh.appendChild(newEle);

And there we go. We have now added the following H6 element to the end of the 1st article element on this page. (remember that the following H6 element is NOT part of the original HTML). We're also going to change the font color of our new h6 element using newEle.style.color = '#961d1d'; to make it stand out on the page.
