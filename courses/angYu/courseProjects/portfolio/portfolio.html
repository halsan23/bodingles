Development Types

HTML / CSS / GRID / FLEX / BOOTSTRAP / JavaScript

HTML

HTML is the framework for the data you want to present. Define your headings, paragraphs, list items and more.

CSS

Cascading Style Sheets provide the styling to make the webpage come to life. This is where you define your pages look and feel.

Grid / Flex

Grid and Flex are two approaches to defining the layout of a webpage. Where everything goes and how it fits with all of the other components.

Bootstrap

Bootstrap is an external framework designed to combine the functionality of CSS, Grid, and Flex to achieve a "responsive" web layout usually achieved through a mobile first design.

JavaScript

JavaScript is the core programming language that runs underneath it all to provide functionality to the website.

Node / Express

Node and Express are web application frameworks, designed for building web applications and APIs.

SQL

Structured Query Language, is used to manage and manipulate relational databases. These are considered structured databases.

MongoDB

MongoDB is a NoSQL database used to store data in non-relational tables. These are considered non-structured databases.
