Using PostgreSQL

In order to use PostgreSQL you must download and install it. The download link is listed above. Just run the installer. During the installation process, it sets the default username to postgres. You will need to supply a password. Write this password down and don't loose it. It will also want to assign the default port of 5432 but you can change this if you wish.

 

Build a Table

The process to build a table in PostgreSQL is very similar to basic SQL.


  • CREATE TABLE capitols (
    • id SERIAL PRIMARY KEY,
      • SERIAL will auto-increment the PRIMARY KEY ID number
    • country VARCHAR(45),
    • capitol VARCHAR(45)
  • )

We're using VARCHAR(45) to set a maximum length for these fields.
