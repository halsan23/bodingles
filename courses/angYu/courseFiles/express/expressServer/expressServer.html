Express Server

So for the most part, we're going to use Express on a server to process request(s) from a client computer. This means we need a JavaScript file running on the server, (eg. index.js), with Express installed through the npm process. So how do actually create this Express server setup? Good question.

The process has six (6) basic steps:

  1. Create a working directory (CWD)
  2. Create an index/js file
  3. Initialize our npm structure (npm init)
  4. Install the Express package (npm i express)
  5. Write the Server application code in index.js

    • JavaScript
    • // import express and set a variable for default express functionality
    • import express from 'express';
    • const app = express();
    • const port = 3000;
    •  
    • // setup listening port
    • // when we run this file, it starts the server
    • app.listen( port, () => {
      • console.log(`Server running on port: ${port}`);
    • });

  6. Start the server (run node index.js)

The server should now be running and "listening" on port: 3000, for request(s) from a client. Notice that we are using import to import express into our server file. So make sure your package.json file includes:

  • "type": "module",
to allow for the use of the import statement.

Now that our server is up and running, how do we actually access the server output, in our browser? Well, simply enter localhost:3000 into the browser address bar. Our locaL server is the "localhost" and we designated it to run on port 3000.

And while we're talking about ports, if you open a Terminal, and enter the command:

  • netstat -ano | findstr "LISTENING"
you will actually get a list of all of the ports that are open and "Listening" on your computer.

And finally, now that our server is up and running, how do we shut it down?

  • press ctrl - c
and the index.js file that we are running, will terminate.
