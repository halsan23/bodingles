API Authorization
Intro

In order to talk about API Authorization we need to understand the difference between Authentication and Authorization.

  • Authentication the process of verifying the identity of a user, device, or system. It ensures that the entity claiming to be who or what it is, actually is who or what it claims to be.
  • Authorization is the process of determining whether an authenticated entity has the right to access a specific resource or perform a specific action.

When using APIs we are usually talking about Authorization. We are going to verify that this user has "Authorization" to access this APIs resources.
