CSS Selectors
Intro

To assign styles to html elements we need to be able to "select" the relevant elements. We do this with CSS Selectors.

Selectors
  • Attribute Selector []
    • A Way to specifically target an element's attribute. Attributes are things such as ID, or Class, or "draggable". If we have the following example:
      • <p id="priority">This paragraph has priority</p>
      we can select it in the CSS file with p[#priority] {} and apply our style. So we are only targeting the <p> element with an ID of priority
  • Group Selector , (comma)
    • Allows us to target multiple elements. If we have an <h1> and an <h2> element that we want to select, we would use:
      • h1, h2 {}
      This would group both elements for styling.
  • Child Selector > (greater than)
    • Allows us to select a child element. If we had a <div> with a child <p> element, we could select the child, like so:
      • div > p {}
      This would select all <p> elements, that are children to a <div> element.
