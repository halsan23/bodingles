Responsive Web Design

Responsive Web Design is about using HTML and CSS to automatically resize, hide, shrink, or enlarge, a website, to make it look good on all devices (desktops, tablets, and phones). In essence a website can adjust itself to look correct on different sized screens, or devices.

There are basically four different tools that we can use to achieve this.

  • Media Queries
  • Flexbox
  • CSS Grid
  • Bootstrap

The first three options are built into CSS. Bootstrap is the only external framework.
