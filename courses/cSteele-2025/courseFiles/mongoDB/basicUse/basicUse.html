MongoDB
Basic Usage

In this discussion, we're going to start talking about how to actually use MongoDB. We're going to be using the MongoDB Shell to create a db, add to a db, and remove from a db.

We're going to create an "Animal Shelter" db and to do this we'll use the keyword use, which is the command to create a new db, followed by the db name. So start by typing (in the Mongo Shell), the following:

  • use animalShelter

and once we hit [Enter], the Mongo Shell displays "switched to db animalShelter", and our prompt has switched to animalShelter> to show us that we are now working within our new db. We've now created our first MongoDB database.

We now need to understand that within Mongo, we work with different objects that are referred to as "Collections". For example, in our Animal Shelter, we might have collections of Dogs, Cats, People, or Locations. We're going to start by creating a "Dog" collection. This is called inserting into the db.

In Mongo, if we try to insert something into a collection that does not yet exist, then that collection is created. And if we look at the Mongo Docs, we'll see that there are two ways to insert data. Either insert one, or insert many. We're going to start by inserting one. Let's show the command and then break it down:

  • db.dogs.insertOne({})
So, while we are in our database (animalShelter), we tell the db to either create, or insert into, the dogs collection and we want to insertOne({}). Now if you notice, insertOne() is looking for an object about the collection we want to insert. So let's actually add a collection of a dog named Chico:
  • db.dogs.insertOne({name: "Chico", age: 3, breed, "Pit Bull", catFriendly: true})
Now the first thing you probably noticed is that this looks an awful lot like a json object, but wait, why are the key names not enclosed in parenthesis? It's because it's a "JavaScript" object, not a json object. So we don't need the parenthesis.

Once we hit enter, mongo returns a promise:

  • {
    • acknowledged: true,
    • insertedId: ObjectId('67f5908cf09a1c0894b71236')
  • }
and if we enter show collections, we see dogs has been created. We can also use db.dogs.find() to display all of the entries in this collection, and we get:
  • [
    • {
      • _id: ObjectId('67f5908cf09a1c0894b71236'),
      • name: 'Chico',
      • age: 10,
      • breed: 'Pit Bull',
      • catFriendly: true
    • }
  • ]
